# Coupe du monde 2026 - Portugal

Objectif de ce fichier :

- Centraliser les infos liées à la Coupe du monde 2026.
- Garder l'itinéraire principal propre.
- Ne pas laisser les matchs piloter le voyage.

## Principe validé

Les matchs du Portugal sont des bonus. Ils peuvent influencer le rythme d'une journée, mais ne doivent pas casser les gros moments du voyage :

- Fuji
- Gion Matsuri
- Nara
- USJ / Super Nintendo World
- Hiroshima / Miyajima
- Dernier jour Tokyo avant départ

Exception possible :
- Demi-finale ou finale du Portugal, à gérer comme moment spécial.

## Contexte

- Le Portugal est dans le groupe K.
- La phase de groupe sera terminée avant le départ au Japon.
- Le dernier match de groupe Colombie - Portugal est prévu le 27 juin 2026.
- Le chemin exact sera connu avant le vol du 03 juillet 2026.

## Stratégie sommeil

Pour les matchs à 04:00 ou 05:00 :

- Dormir vers 21:30-22:00.
- Réveil 15 minutes avant match.
- Match.
- Redormir de 06:00/06:30 à 09:00/10:00.
- Alléger la journée.

Ne pas cumuler :

- Match à 04:00.
- Grosse marche extérieure.
- Chaleur de midi.
- Soirée tardive.

## Où regarder

### Avion Air France

- Air France déploie le Wi-Fi Starlink gratuit, mais ce n'est pas garanti sur chaque avion.
- Si le Portugal termine 1er du groupe, le 1/16e tombe pendant le vol CDG -> HND.
- Prévoir que ce match peut être seulement suivi en live texte / résumé.

### Tokyo

Options à vérifier après le 27 juin :

- Sports bars à Shibuya / Ebisu / Roppongi.
- HUB si programmation spéciale.
- Hôtel comme fallback, surtout pour horaires très tôt.

### Kyoto

Options à vérifier :

- HUB Kyoto.
- Bars autour de Sanjo / Kawaramachi.
- Hôtel fallback pour matchs à 04:00.

Point important :
- Demi potentielle pendant Gion Matsuri. Si match à 04:00 le 16 juillet, la journée doit rester light pour profiter de Yoiyama.

### Osaka

Meilleure base pour une éventuelle finale :

- Namba / Shinsaibashi / Umeda.
- HUB Namba Ebisubashi.
- HUB Umeda.
- Sports bars internationaux.

Finale potentielle :
- Lundi 20 juillet 2026 à 04:00 JST.
- Journée Osaka chill déjà prévue, donc c'est le meilleur cas pour intégrer un très gros match.
