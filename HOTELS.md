# Hôtels

## Objectif budget

Plafond moyen à viser :

- Objectif : ¥25'000-28'000 par nuit pour 2.
- Maximum normal : ¥30'000 par nuit pour 2.
- Kyoto pendant Gion Matsuri : tolérance jusqu'à ¥35'000 par nuit.
- Nuit spéciale Fuji : tolérance jusqu'à ¥55'000-75'000 pour 2.

Plafond total hôtel conseillé :

- Cible : ¥580'000-650'000 pour 23 nuits avec Fuji.
- Zone haute acceptable : ¥700'000 si les emplacements sont vraiment bons.
- Au-delà : risque de faire exploser le budget global.

## Zones recommandées

### Tokyo début

Zones :
- Shinjuku
- Shibuya
- Ebisu

Pourquoi :
- Bon accès vers Kawaguchiko/Fuji depuis Shinjuku.
- Quartiers vivants pour les premières soirées.
- Connexions faciles dans Tokyo.

À surveiller :
- Chambre trop petite.
- Trop loin d'une station.
- Prix qui monte le week-end.

### Kawaguchiko / Fuji

Zones :
- Nord du lac Kawaguchiko.
- Fujikawaguchiko avec vue Fuji.
- Shimoyoshida si priorité Chureito Pagoda tôt le matin.

Pourquoi :
- Meilleure chance de voir Fuji très tôt.
- Expérience couple : lac, onsen, dîner calme.

Critères importants :
- Vue Fuji si possible.
- Annulation gratuite ou conditions souples.
- Accès navette / taxi / bus.
- Possibilité de petit-déjeuner tôt ou flexible.

### Sendai

Zone :
- Autour de Sendai Station.

Pourquoi :
- Pratique pour Ishinomaki / Tashirojima.
- Simple pour repartir vers Kyoto.
- Pas une étape où il faut payer cher.

### Kyoto

Zones :
- Shijo-Karasuma
- Kawaramachi
- Sanjo
- Gion-Shijo si prix acceptable

Pourquoi :
- Meilleures zones pour Gion Matsuri.
- Retour facile le soir.
- Accès restos, métro/bus, Nishiki, Gion.

À surveiller :
- Prix Gion Matsuri.
- Bruit / foule.
- Annulation.

### Osaka

Zones :
- Namba / Shinsaibashi recommandé.
- Umeda si priorité transport.

Pourquoi :
- Namba/Shinsaibashi est plus fun en couple.
- Dotonbori et nourriture proches.
- USJ reste accessible.

### Hiroshima

Zones :
- Hatchobori / Hondori recommandé.
- Hiroshima Station si priorité transport.

Pourquoi :
- Centre-ville plus agréable le soir.
- Accès Peace Park et restaurants.
- Station pratique pour Miyajima et retour Tokyo.

### Tokyo fin

Zones :
- Shinjuku/Shibuya si backup Fuji prioritaire.
- Ginza/Shimbashi si départ Haneda prioritaire.

Pourquoi :
- Dernier bloc flexible.
- Shopping et départ simple.
