## Objectif budget

Plafond moyen a viser :

- Objectif hors Fuji : ¥25'000-30'000 par nuit pour 2.
- Kyoto pendant Gion Matsuri : tolerance jusqu'a ¥35'000 par nuit.
- Fuji : tolerance jusqu'a ¥55'000-75'000 par nuit pour 2.
- Tokyo fin : tolerance jusqu'a ¥32'000 par nuit si l'emplacement facilite Haneda.

Plafond total hotel conseille :

- Cible optimiste : ¥680'000-740'000 pour 23 nuits avec 2 nuits Fuji.
- Zone haute acceptable : ¥740'000-820'000 si Fuji/Kyoto/Tokyo sont bien places.
- Au-dela : risque de faire exploser le budget global.

## Zones d'hebergement validees

Methode de recherche :

- Chercher d'abord dans la liste 1, sans trop regarder le prix.
- Si les prix sont trop eleves, si les chambres sont trop petites ou si les conditions d'annulation sont mauvaises, passer a la liste 2.
- Dans tous les cas, viser un hotel a moins de 10 minutes a pied d'une station utile.
- Prioriser le sommeil : eviter les zones tres bruyantes meme si elles sont centrales.

### Liste 1 - Meilleures regions sans tenir compte du prix

| Base | Zone ideale | Pourquoi ce choix |
|---|---|---|
| Tokyo debut | Shinjuku Sud / Yoyogi | Bon compromis pour 7 nuits : connexions solides, acces simple a Shibuya/Harajuku/Shinjuku, et depart Fuji plus pratique via Shinjuku. Plus calme que Kabukicho profond. |
| Kawaguchiko / Fuji | Nord / nord-est du lac Kawaguchi, secteur Asakawa | Meilleure zone pour l'experience Fuji : vue potentielle, ambiance lac, ryokan/onsen, calme le soir. Avec 2 nuits, l'emplacement compte vraiment. |
| Kyoto | Shijo-Kawaramachi / Kawaramachi-Sanjo | Base globale efficace pour Gion, Pontocho, Nishiki, les soirees et Gion Matsuri. Vivant sans etre trop eloigne des visites. |
| Osaka | Namba / Shinsaibashi | Meilleur choix pour profiter d'Osaka sur 5 nuits : Dotonbori, food, soirees, Namba Yasaka-jinja. USJ et Hiroshima restent faisables avec un peu d'organisation. |
| Hiroshima | Hondori / Hatchobori | Meilleur equilibre entre visites, restos et transport local. Peace Memorial Park est facile, Hondori donne acces au centre, et les trams gerent gare/Miyajima. |
| Tokyo fin | Ginza / Shimbashi | Bon final : shopping, restos, ambiance plus calme, acces simple a Tokyo Station et Haneda. Pratique apres le retour depuis Hiroshima. |

Points d'attention pour la liste 1 :

- Tokyo debut : eviter Kabukicho profond si l'objectif est de bien dormir.
- Fuji : verifier navette depuis Kawaguchiko Station, diner sur place ou restaurants accessibles, car les bus touristiques s'arretent assez tot.
- Kyoto : Gion pur peut etre plus cher, plus touristique et parfois moins pratique que Kawaramachi/Sanjo.
- Osaka : Namba maximise l'ambiance ; si le confort transport devient prioritaire, Umeda peut repasser devant.
- Hiroshima : eviter de dormir trop loin des trams si l'objectif est Hiroshima + Miyajima en deux jours.
- Tokyo fin : Hamamatsucho est plus pratique pour Haneda, mais Ginza/Shimbashi est plus agreable pour finir.

### Liste 2 - Regions backup moins cheres

| Base | Zone backup moins chere | Pourquoi ce choix |
|---|---|---|
| Tokyo debut | Nishi-Shinjuku / Higashi-Shinjuku / Ueno | Zones bien connectees, parfois moins cheres. Ueno peut etre malin pour Asakusa/Akihabara, mais moins direct pour Fuji. |
| Kawaguchiko / Fuji | Kawaguchiko Station / Kawaguchiko Ohashi | Moins magique que le nord du lac, mais plus simple sans voiture : bus, taxis, restaurants, arrivees/departs. |
| Kyoto | Shijo-Karasuma / Karasuma-Gojo | Bon rapport praticite/prix/sommeil. Shijo-Karasuma reste proche du Gion Matsuri ; Karasuma-Gojo baisse souvent les prix. |
| Osaka | Umeda / Honmachi / Fukushima | Umeda est meilleur pour les trains, USJ et Hiroshima ; Honmachi est central ; Fukushima est plus calme et souvent cool pour les restos. |
| Hiroshima | Hiroshima Station / Ekimae | Moins sympa le soir que Hondori/Hatchobori, mais imbattable pour le shinkansen et les connexions JR. |
| Tokyo fin | Hamamatsucho / Daimon | Moins "waouh" que Ginza/Shimbashi, mais tres pratique pour Haneda via monorail et bon choix anti-stress. |

Zones a eviter sauf deal exceptionnel :

- Tokyo debut : hotels loin d'une station ou au coeur de Kabukicho si le sommeil est prioritaire.
- Fuji : hotels isoles sans navette, sans diner et sans acces taxi clair.
- Kyoto : Arashiyama, Fushimi, nord de Kyoto ou pentes de Kiyomizu comme base de 5 nuits.
- Osaka : Universal City pour tout le sejour, car pratique seulement pour USJ.
- Hiroshima : Miyajimaguchi pour les 2 nuits, car trop excentre pour Hiroshima ville.
- Tokyo fin : Shinjuku/Shibuya si la priorite est le depart Haneda et le shopping final plus calme.
